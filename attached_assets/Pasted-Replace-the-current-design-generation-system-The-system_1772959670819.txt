## CORE PROBLEM

The generator currently changes only surface properties such as:

colors
rounded corners
spacing
minor component styling

This means the underlying layout architecture remains identical.

The generator must instead produce **entirely different page structures**, not style variations.

## NEW GENERATION MODEL

Implement a **Structural Layout Generator** that builds interfaces from a layout grammar instead of templates.

Create:

/ai/layout/structuralGenerator.ts

The generator must build pages using layout primitives rather than templates.

Layout primitives include:

vertical stack
split grid
asymmetric grid
sidebar layout
centered hero
left-heavy hero
right-heavy hero
masonry grid
content bands
floating panels
dashboard clusters
full-width sections

The system must combine these primitives into new page structures.

## STRUCTURE FIRST PIPELINE

Replace the current generation flow.

Old flow:

prompt → template → style adjustments

New flow:

prompt
→ context reasoning
→ layout structure generation
→ component placement
→ style generation

Structure must be generated **before** styling.

## LAYOUT DNA SYSTEM

Create:

/ai/layout/layoutDNA.ts

Each project must generate a layout DNA object.

Example structure:

{
heroType,
sectionOrder,
gridStructure,
navigationType,
contentDensity,
componentGrouping,
visualHierarchy
}

The layout generator must use this DNA to build the page.

No two projects may share identical layout DNA.

Create:

/ai/layout/structureEntropy.ts

It must randomize:

section order

grid column counts
content band arrangements
component placement
navigation structure

This randomness must remain compatible with the prompt context.

Create:

/ai/components/architectureVariants.ts

file browser grid
file browser table

file card gallery
column-based storage layout
panel-based storage interface

The generator must select among these architectures using layout DNA.

Create:

/ai/layout/layoutSimilarity.ts

compare layout DNA with previous projects.

regenerate layout DNA.
