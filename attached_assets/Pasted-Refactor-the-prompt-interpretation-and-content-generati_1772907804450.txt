------------------------------------------------
1. SEMANTIC PROMPT INTERPRETER
------------------------------------------------

Create a semantic interpretation layer.

File:
/lib/semanticInterpreter.ts

Input:
raw user prompt

Output structured context:

{
industry
productType
companyType
targetAudience
services
tone
keywords
semanticDomain
}

Example

Prompt:
"landing page for a fintech startup managing payroll for remote teams"

Interpreter output:

industry: fintech
productType: payroll software
companyType: startup
targetAudience: companies with distributed teams
services:
- payroll automation
- compliance
- employee payments
tone: modern fintech
semanticDomain: finance technology

This must be generated using semantic reasoning, not keyword detection.

------------------------------------------------

Content generator must only produce words belonging to:

------------------------------------------------

Every generated sentence must pass a relevance score.

File:
/lib/relevanceScore.ts
